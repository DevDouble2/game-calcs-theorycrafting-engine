## 🛠️ Technical Overview
This repository drives the simulation logic and data structures for **GameCalcs**. We prioritize mathematical accuracy and real-time game data synchronization:

* **ARPG Simulation Engine**: High-fidelity calculators for **Path of Exile 2 (PoE2)**, including Temple Planners and Recombinator odds.
* **Logic-Based Breeding Patterns**: Authentic CombiRank formula implementations for **Palworld** and other creature-collection titles.
* **Interactive Skill Mapping**: Complex prerequisite tracking systems for **ARC Raiders** and **Fallout** character builders.
* **Market Data Integration**: Real-time pricing logic for **Pokemon TCG** collectors via TCGplayer/eBay API wrappers.

### Key Features
- **Precision Theorycrafting**: Move beyond "feel-based" builds to deterministic mathematical models.
- **Cross-Platform PWA**: High-performance UI built with Next.js, optimized for second-screen use during gameplay.
- **Hardcore-First UX**: Distraction-free, data-rich interfaces designed for elite players.
